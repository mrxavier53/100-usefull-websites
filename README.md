## ✨ What is this?
A GitHub Pages ready animated directory of 100 useful websites, grouped into categories, now with a green Legendary Labs theme while keeping the simple original-style layout.

## 🚀 GitHub Pages Setup
1. Upload all files to your GitHub repository.
2. Go to **Settings → Pages**.
3. Select the branch and root folder.
4. Open the deployed URL.
5. Use the category sections to find useful links quickly.

## 🛠 Editing Guide
- Edit `index.html` for page text and layout.
- Edit files inside `assets/css/` for colors and styling.
- Edit files inside `assets/js/` for app logic and interactions.
- Keep paths relative so GitHub Pages works properly.
